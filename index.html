Tools

HTML

HTML stands for HyperText Markup Language. It's the standard language used to create and structure content on the web. HTML is the building block of web pages and is used to define the structure and layout of a webpage's content, including text, images, links, and multimedia elements.

CSS

CSS stands for Cascading Style Sheets. It's a stylesheet language used in web development to control the presentation and layout of HTML elements on a webpage. CSS allows you to define how the content of a webpage should look and be positioned, separate from the actual content itself.

BOOTSTRAP

Bootstrap is a popular open-source front-end framework for building responsive and visually appealing websites and web applications. It was originally developed by Twitter and is now maintained by a community of developers. Bootstrap provides a collection of HTML, CSS, and JavaScript components, templates, and tools that make it easier to create consistent and well-designed user interfaces.
